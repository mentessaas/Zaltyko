# Archivo normativo

Esta carpeta conserva documentos oficiales usados como fuente primaria para los catalogos
deportivos de Zaltyko. No forma parte del runtime ni sustituye la validacion de una persona
especialista cuando una regla federativa sea ambigua.

## Espana / RFEG 2026

Los PDF de `España/` fueron proporcionados por el usuario y contrastados el 2026-07-12.
Sustentan la nomenclatura GAF, GAM y GR versionada como `rfeg-2026-v2` en
`src/lib/sport-config/catalog.ts`.

Pendientes conocidos:

- el programa Base de GR no esta confirmado por estos documentos;
- la relacion categoria-aparato de GR aun no esta modelada en el schema;
- los seeds legacy de `src/db/seeds/templates/` no son la fuente canonica del catalogo.
