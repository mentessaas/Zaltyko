# Onboarding de cliente

## Checklist

- Crear academia.
- Configurar datos basicos y deporte.
- Cargar atletas.
- Crear clases/grupos.
- Configurar coaches.
- Activar billing si aplica.
- Registrar primera asistencia o cobro.
- Revisar dashboard con cliente.

## Exito del onboarding

El cliente debe poder operar una semana real con menos dependencia de Excel/WhatsApp.

## Senales de riesgo

- No carga atletas.
- No crea clases.
- No invita staff.
- No entiende pricing/cobros.
- Pide features no implementadas como condicion para continuar.
